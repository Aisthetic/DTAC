The Program include some options when using as a CLI tool (in command line)
You can use the software with these arguments

	-i <path-to-proxy-txt-file>	The default is proxy.txt, but you can use this argument
					to alter the default
	-o <path-to-output-file>	The default is accountInfo.txt, use this argument to alter.

	--no-proxy			Disables using account creation
